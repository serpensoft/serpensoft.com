    4.14

    • The issue of incorrect editor insets after device rotation has been resolved.
    • The bug that prevented additional settings options from being displayed in System Settings has been fixed.
    • Mermaid plugin updated (11.6.0).
    • Internal application links updated.
    • Introduction documents updated.
    • Other minor bug fixes.

    4.13.1

    • Mermaid plugin updated (11.4.1).
    • Other minor bug fixes.

    4.12

    • Fixed issue with printing of long ordered lists.
    • Fixed issues with plain text printing.
    • Mermaid plugin updated (10.9.1).
    • Flowchart plugin updated (1.18).
    • Other minor bug fixes.

    4.11.3

    • Fixed several bugs with line shift actions.
    • Mermaid plugin updated (10.9.0).
    • Minor bug fixes and improvements.
    • iWriter Pro now requires iOS 16 or higher.

    4.11.2

    • Mermaid plugin updated (10.7.0).
    • Minor bug fixes and improvements.
    • iWriter Pro now requires iOS 16 or higher.

    4.11

    • Added page margins menu for printing and PDF export.
    • Added page header/footer menus for printing and PDF export.
    • Fixed issues with documents containing restricted ASCII symbols and HTML entities when exporting to EPUB and DOCX.
    • Fixed issues when printing Markdown text.
    • Mermaid plugin updated (10.6.1).
    • Highlight.js plugin updated (11.9.0).
    • iWriter Pro now requires iOS 16 or higher.

    4.10.0

    • Improvements to PDF export menu.
    • Fixed issues with DOCX export for documents containing HTML entities.
    • Non-focused text color in Night and Dark themes became lighter.
    • Mermaid plugin updated (10.5.0).

    4.9.9

    • Fixed bug in Preview which prevented opening external links.
    • Highlight.js plugin updated (11.8.0).
    • Mermaid plugin updated (10.4.0).

    4.9.8

    • Mermaid plugin updated (10.2.4).
    • Fixed some minor issues with localisations.
    • Resolved some minor UI issues.

    4.9.7

    • Document now correctly updates its state if a Todo item is clicked.
    • Mermaid plugin updated (9.4.3). This update includes support for Mindmap and Timeline diagrams.
    • Resolved some minor UI issues.

    4.9.6

    • Fixed bug with incorrect equation numbering.
    • Highlight.js plugin updated (11.7.0).
    • Mermaid plugin updated (9.3.0).
    • iWriter Pro now requires iOS 15.6 or higher.

    4.9.5

    • Added new Access Issues menu for images and files located outside app’s sandbox.
    • Highlight.js plugin updated (11.6.0).
    • Mermaid plugin updated (9.1.7).
    • Fixed issues with keyboard shortcuts on iOS 16.
    • Other minor bug fixes and improvements.

    4.9.4

    • Added new Access Issues menu for images and files located outside app’s sandbox.
    • Application will now display a notification for files that cannot be loaded due to sandbox restrictions.
    • Mermaid plugin updated (9.1.7).
    • Other minor bug fixes and improvements.
    • iWriter Pro now requires iOS 15 or higher.

    4.9.3

    • Preview now supports text alignment option.
    • User interface improvements for Search bar and Preview options.
    • Home indicator now automatically disappears on iPad.
    • Mermaid plugin updated (9.1.5).
    • Other minor bug fixes and improvements.
    • iWriter Pro now requires iOS 15 or higher.

    4.9.2

    • Mermaid code blocks now support syntax highlighting.
    • Syntax highlighting for LaTeX and HTML was updated.
    • Added new option for disabling code syntax highlight in editor settings.
    • Preview plugins updated: MathJax (3.2.2), Mermaid (9.1.3).
    • Fixed crash in files containing multiple strikethrough style elements.
    • Fixed issues with exporting files to EPUB.
    • Other minor bug fixes and improvements.

    4.9

    • Mermaid code blocks now support syntax highlighting.
    • Syntax highlighting for LaTeX and HTML was updated.
    • Added new option for disabling code syntax highlight in editor settings.
    • Preview plugins updated: MathJax (3.2.2), Highlight.js (11.5.1).
    • Fixed issue with Search & Replace not working for code blocks.
    • Fixed crash in files containing multiple strikethrough style elements.
    • Other minor bug fixes and improvements.

    4.8.2

    • Preview scroll sync should be more consistent for different documents.
    • New “Insert Footnote” keyboard shortcut added.
    • Other keyboard shortcuts have been updated.
    • Preview plugins updated: Mermaid (9.1.1), Highlight.js (11.5.1).
    • Other minor bug fixes and improvements.

    4.8

    • Added syntax highlighting for inline HTML.
    • Added syntax highlighting for LaTeX equations.
    • Preview now synchronizes its scroll position with the Editor.
    • Performance improvements for Preview and Editor.
    • Tweaked colors for math, code blocks, and other elements.
    • Preview plugins updated: Mermaid (8.14), Flowchart (1.17).
    • Fixed various issues with Markdown highlighting.
    • Fixed bugs causing inconsistent text field size.
    • Fixed “Copy HTML” export action.
    • Fixed issues with paper size selection.
    • Fixed various bugs in DOCX and EPUB export.
    • Other minor bug fixes and improvements.

    4.7

    • iPhone keyboard row now features undo/redo buttons.
    • Improved highlighting for images and links.
    • Tweaked colors for Light and Dusk themes.
    • Preview plugins updated: Mermaid (8.13.4), Flowchart (1.17.0), Highlight.js (11.3.1).
    • Other bug fixes and improvements.

    4.6.6

    • Tweaked colors for dark themes.
    • Fixed a rare bug with unresponsive text editor.
    • Fixed issues with export to DOCX and LaTeX formats.
    • Fixed highlighting for strikethrough elements containing sub-items.
    • Fixed highlighting for multi-line Math elements.
    • Handoff issues resolved.
    • Other minor bug fixes and improvements.

    4.6.5

    • Fixed issues with export to DOCX format.
    • Fixed issues with export to LaTeX format.
    • Fixed highlighting for strikethrough elements containing sub-items.
    • Fixed highlighting for multi-line Math elements.
    • Fixed a bug with Handoff feature.
    • Other minor bug fixes and improvements.

    4.6

    • New QuickLook plugin for file thumbnails.
    • Fixed issues with the behavior of paragraph spacing option.
    • Fixed highlighting for strikethrough elements containing sub-items.
    • Fixed highlighting for multi-line Math elements.
    • Mermaid plugin updated to the version 8.12.0.
    • Other minor bug fixes and improvements.
