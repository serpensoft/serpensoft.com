    • Resolved various bugs and user-interface issues.

    • Added new settings options for line height, paragraph spacing, and indentation.
    • List insertion shortcuts can now be applied to several lines at once.
    • Added option for displaying code line numbers in HTML preview.
    • Colors and HTML preview styles for code blocks updated.
    • QuickLook plugin updated with new style for code blocks.å
    • Resolved various user-interface issues.

    • MathJax plugin updated to version 3.1.2.
    • Fixed issue with System font being unavailable in settings.
    • App icon updated.
